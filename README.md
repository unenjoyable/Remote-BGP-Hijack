# Remote-BGP-Hijack

Remote-BGP-Hijack is a Python script for hijacking bgp servers.

## Installation

first download pip2 using 
```bash
wget https://bootstrap.pypa.io/get-pip.py
python2 get-pip.py
```
## Usage

```
python2 bgp.py (ip)
```
## Scaning ranges

```
zmap (ip)/24 -p 179
```
